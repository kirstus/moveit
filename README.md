# Move it app

App desenvolvido na trilha de React durante a Next Level Week ([NLW#4](https://nextlevelweek.com/))

Baseado na **técnica Pomodoro**, o app utiliza ciclos de produtividade de 25 minutos, e ao final de cada ciclo o usuário recebe um desafio de bem estar digital para reduzir os efeitos colaterais que o uso prolongado de computador pode causar na visão e na musculatura.

Contém elementos de gamificação, exibindo um level para o usuário baseado no número de desafios completos e uma barra de pontos de experiência necessários para o próximo level.
